# About this project

This is a simple banking app built with Laravel, its designed to manage and organize customer information. Its a fully functional CRUD, having the ability to create new customers, view certain customer while editing and deleting their profiles. The create, edit and delete features are locked behind admin authentication while the regular users can only view the customers profiles.

# Features

-   CRUD Functionality
-   Admin Authentication
-   Customer details search filtering

## How to run the project

1. Configure .env with database details
2. Run 'npm install' and 'composer install' to download all packages
3. Run 'npm run dev' and 'php artisan serve' to start the development server
4. Open the local host and register a new user
